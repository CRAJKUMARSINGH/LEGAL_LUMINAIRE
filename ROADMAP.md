# Roadmap (Q2 2026)

## P0 — Reliability baseline
- CI: typecheck + build + backend syntax checks (GitHub Actions)
- Security audits: pnpm audit + pip-audit (scheduled + PR)
- One-command local: Docker Compose + docs

## P1 — Demo polish
- Screenshot/GIF section for repo landing page
- Demo mode navigation improvements + clearer synthetic labeling
- Publish a frontend-only static demo (Netlify/Vercel) with disclaimers

## P2 — Accuracy & evidence controls
- Document upload: connect UI upload to backend indexing end-to-end
- Stronger “primary source” emphasis in UI (what came from uploads vs web)
- Expand contradiction detection beyond dates (names, amounts, locations)

## P3 — Observability and operations
- Request tracing (agent-by-agent)
- Token/cost reporting per case/session
- Rate limiting for expensive endpoints
