## GENERATION STATUS

These outputs are STATIC (pre-generated) — they do not require OpenAI API.
They demonstrate what the app produces when fully configured.

To generate fresh outputs with real API:
1. Configure `OPENAI_API_KEY` in `.env`
2. Start backend: `artifacts/legal-luminaire/backend/start.bat`
3. Load Case 01 via `/intake`
4. Navigate to `/discharge-application`
