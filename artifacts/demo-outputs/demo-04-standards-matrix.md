The prosecution applied IS 1199:2018 to analyse hardened cement mortar
from existing stone masonry. This is a foundational scientific error:

IS 1199:2018 governs FRESH CONCRETE — it has no application to hardened
masonry mortar sampled from a structure that has been standing for years.

The correct standards are:
- IS 2250:1981 for masonry mortar composition
- ASTM C1324 for forensic analysis of hardened masonry mortar

Since the prosecution's FSL report is based on the wrong standard, the
entire forensic analysis is scientifically invalid and must be excluded.
