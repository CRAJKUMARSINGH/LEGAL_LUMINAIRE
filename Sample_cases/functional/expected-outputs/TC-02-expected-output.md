# TC-02 — Expected Output Specification
## NDPS Bail Application

---

## DOCUMENT TYPE
Bail Application under §439 CrPC / §483 BNSS

## LANGUAGE
Hindi (Devanagari script)

## MANDATORY SECTIONS
- [ ] Court heading
- [ ] Party names
- [ ] Case number and charges (NDPS §8/21/29)
- [ ] Facts section
- [ ] Grounds — minimum 5 numbered grounds
- [ ] §52A non-compliance as Ground 1 (primary)
- [ ] Precedents — minimum 2 citations
- [ ] Prayer clause — specific bail conditions
- [ ] Verification report
- [ ] Pre-filing checklist

## MANDATORY CONTENT CHECKS
- §52A NDPS must be cited as primary ground
- "No Magistrate present at sampling" must be stated
- FSL delay (45 days vs 30-day limit) must be stated
- No independent witness must be stated

## BLOCKED CONTENT
- Any PENDING citation in draft body
- §52A cited as complied with (factually wrong)
