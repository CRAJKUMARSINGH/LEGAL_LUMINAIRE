# STRICT ACCURACY RULES - LEGAL LUMINAIRE
## MANDATORY GUIDELINES FOR ALL FUTURE UPDATES

### 🎯 **CORE ACCURACY PRINCIPLE**
**100% VERIFICATION REQUIRED** - No unverified claims, no assumptions, no invented content.

### 📋 **STRICT ACCURACY CHECKLIST**

#### 1. **LEGAL CITATIONS**
- [ ] Every case law must have verified citation
- [ ] Exact paragraph numbers required for Supreme Court judgments
- [ ] Secondary sources marked as "SECONDARY - verification pending"
- [ ] No invented quotes or paraphrases
- [ ] Cross-reference with official court databases

#### 2. **STANDARDS & CODES**
- [ ] IS/ASTM/BIS codes must reference exact clause numbers
- [ ] No "approximate" or "likely" language
- [ ] Official standard documents as primary sources
- [ ] Version numbers and dates mandatory

#### 3. **FACTUAL ASSERTIONS**
- [ ] Every fact must be traceable to source document
- [ ] No speculation presented as fact
- [ ] Clear distinction between verified vs pending verification
- [ ] Source references in footnotes/endnotes

#### 4. **FORENSIC CLAIMS**
- [ ] Laboratory procedures must follow documented protocols
- [ ] Chain of custody requirements strictly verified
- [ ] No technical claims without scientific basis
- [ ] Expert opinions must cite exact methodology

#### 5. **PROCEDURAL REQUIREMENTS**
- [ ] Court procedures verified with latest rules
- [ ] BNSS 2023 compliance mandatory
- [ ] No outdated procedural references
- [ ] Local court rules verified where applicable

### ⚠️ **ZERO-TOLERANCE VIOLATIONS**
- **Invented case law** → Immediate rejection
- **Unverified standards** → Immediate rejection
- **Assumed facts** → Immediate rejection
- **Speculative legal opinions** → Immediate rejection

### 🔄 **VERIFICATION WORKFLOW**
1. **Source Verification** → Official court/standards websites
2. **Cross-Reference** → Multiple independent sources
3. **Document Trail** → Save all verification sources
4. **Peer Review** → Second verification required
5. **Final Approval** → Only then integrate into system
